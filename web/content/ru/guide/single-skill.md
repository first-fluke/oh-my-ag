# Сценарий: Одиночный навык

## Когда использовать этот путь

Используйте, когда результат имеет узкий объём и в основном принадлежит одному домену:

- один UI-компонент
- один API-эндпоинт
- одна ошибка в одном слое
- один рефакторинг в одном модуле

Если задача требует кросс-доменной координации (API-контракт + UI + QA), используйте [`Мультиагентный проект`](./multi-agent-project.md).

## Предпусковой чек-лист

Перед формулировкой промпта определите:

1. точный результат (файл или поведение)
2. целевой стек и версии
3. критерии приёмки
4. ожидания по тестированию

## Шаблон промпта

```text
Build <specific artifact> using <stack>.
Constraints: <style/perf/security constraints>.
Acceptance criteria:
1) ...
2) ...
Add tests for: <critical cases>.
```

## Пример промпта

```text
Create a login form component in React + TypeScript + Tailwind CSS.
Constraints: accessible labels, client-side validation, no external form library.
Acceptance criteria:
1) email and password validation messages
2) disabled submit while invalid
3) keyboard and screen-reader friendly
Add unit tests for valid/invalid submit paths.
```

## Ожидаемый процесс выполнения

1. Подходящий навык выбирается автоматически.
2. Агент предлагает реализацию и допущения.
3. Вы подтверждаете или корректируете допущения.
4. Агент предоставляет код и тесты.
5. Вы проводите локальную верификацию и запрашиваете небольшие доработки.

## Контрольная проверка перед слиянием

- поведение соответствует критериям приёмки
- тесты покрывают основной сценарий и ключевые граничные случаи
- нет несвязанных изменений файлов
- нет скрытых обратно несовместимых изменений в общих модулях

## Сигналы эскалации

Переходите к мультиагентному процессу, когда:

- работа над UI требует новых API-контрактов
- одно исправление вызывает каскадные изменения между слоями
- объём выходит за пределы одного домена после первой итерации

## Критерии завершения

Выполнение одиночного навыка завершено, когда:

- целевой артефакт реализован
- критерии приёмки доказуемо выполнены
- тесты добавлены или обновлены для изменённого поведения
