# Введение

oh-my-ag — это мультиагентный оркестратор для Antigravity IDE. Он маршрутизирует запросы к специализированным навыкам и координирует агентов через память Serena.

## Что вы получаете

- Маршрутизация навыков на основе запроса
- Выполнение на основе рабочих процессов для планирования/ревью/отладки
- CLI-оркестрация для параллельного запуска агентов
- Панели мониторинга в реальном времени для отслеживания сессий

## Роли агентов

| Агент | Ответственность |
|---|---|
| workflow-guide | Координация сложных мультидоменных проектов |
| pm-agent | Планирование и архитектурная декомпозиция |
| frontend-agent | Реализация на React/Next.js |
| backend-agent | Реализация API/базы данных/авторизации |
| mobile-agent | Реализация на Flutter/мобильная разработка |
| qa-agent | Ревью безопасности/производительности/доступности |
| debug-agent | Анализ корневых причин и безопасные исправления без регрессий |
| orchestrator | CLI-оркестрация субагентов |
| commit | Рабочий процесс конвенциональных коммитов |

## Структура проекта

- `.agent/skills/`: определения навыков и ресурсы
- `.agent/workflows/`: явные команды рабочих процессов
- `.serena/memories/`: состояние оркестрации во время выполнения
- `cli/cli.ts`: источник истины для интерфейсов команд

## Прогрессивное раскрытие

1. Определение намерения запроса
2. Загрузка только необходимых ресурсов навыка
3. Выполнение специализированными агентами
4. Верификация и итерация через циклы QA/отладки
