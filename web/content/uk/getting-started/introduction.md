# Вступ

oh-my-ag -- це мульти-агентний оркестратор для Antigravity IDE. Він маршрутизує запити до спеціалізованих навичок та координує агентів через пам'ять Serena.

## Що ви отримуєте

- Маршрутизація навичок на основі запитів
- Виконання на основі робочих процесів для планування/рецензування/дебагу
- CLI-оркестрація для паралельного запуску агентів
- Панелі моніторингу в реальному часі для відстеження сесій

## Ролі агентів

| Агент | Відповідальність |
|---|---|
| workflow-guide | Координує складні мультидоменні проєкти |
| pm-agent | Планування та архітектурна декомпозиція |
| frontend-agent | Реалізація React/Next.js |
| backend-agent | Реалізація API/бази даних/авторизації |
| mobile-agent | Реалізація Flutter/мобільних додатків |
| qa-agent | Рецензування безпеки/продуктивності/доступності |
| debug-agent | Аналіз першопричин та безпечне виправлення регресій |
| orchestrator | CLI-оркестрація під-агентів |
| commit | Робочий процес конвенційних комітів |

## Структура проєкту

- `.agent/skills/`: визначення навичок та ресурси
- `.agent/workflows/`: явні команди робочих процесів
- `.serena/memories/`: стан оркестрації під час виконання
- `cli/cli.ts`: джерело істини для інтерфейсів команд

## Поступове розкриття

1. Визначити намір запиту
2. Завантажити лише необхідні ресурси навички
3. Виконати за допомогою спеціалізованих агентів
4. Перевірити та ітерувати через цикли QA/debug
