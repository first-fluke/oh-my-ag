# はじめに

oh-my-ag は Antigravity IDE 向けのマルチエージェントオーケストレーターです。リクエストを専門スキルにルーティングし、Serena メモリを通じてエージェントを連携させます。

## 提供される機能

- リクエスト駆動のスキルルーティング
- 計画/レビュー/デバッグのためのワークフローベース実行
- エージェントの並列実行のための CLI オーケストレーション
- セッション監視のためのリアルタイムダッシュボード

## エージェントの役割

| エージェント | 責務 |
|---|---|
| workflow-guide | 複雑なマルチドメインプロジェクトの調整 |
| pm-agent | 計画とアーキテクチャ分解 |
| frontend-agent | React/Next.js の実装 |
| backend-agent | API/データベース/認証の実装 |
| mobile-agent | Flutter/モバイルの実装 |
| qa-agent | セキュリティ/パフォーマンス/アクセシビリティレビュー |
| debug-agent | 根本原因分析とリグレッションセーフな修正 |
| orchestrator | CLI ベースのサブエージェントオーケストレーション |
| commit | コンベンショナルコミットワークフロー |

## プロジェクト構成

- `.agent/skills/`: スキル定義とリソース
- `.agent/workflows/`: 明示的なワークフローコマンド
- `.serena/memories/`: ランタイムオーケストレーション状態
- `cli/cli.ts`: コマンドインターフェースの正式なソース

## 段階的開示

1. リクエストの意図を特定
2. 必要なスキルリソースのみを読み込み
3. 専門エージェントで実行
4. QA/デバッグループで検証と反復
